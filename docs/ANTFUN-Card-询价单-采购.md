## 0. 一句话产品说明(给供应商)

一款**双界面(接触 + 非接 NFC)JavaCard 智能卡**,卡内运行我们自研的钱包/身份 applet(基于开源 Status Keycard)。私钥在安全芯片内生成、不出芯片。需要:GlobalPlatform 可加载 applet 的空白/未个人化卡,或由卡厂完成 applet 装载 + 个人化的成品卡。

## A. 立刻打样(现在下单 · 数量:各 2–3 张)

目的:装我们的 applet 到真卡,验证算法与真机 NFC。**优先选已知开放、无 NDA、GP 可加载的渠道:**

| 项 | 规格 | 来源建议 |
|---|---|---|
| NXP JCOP4 开发卡 | **NXP J3R200(JCOP4 P71,JavaCard 3.0.5,EAL6+,双界面)** | Keycard 官方开发卡(keycard.tech,同芯片家族,已确认未熔断/可加载);Satochip DIY 卡(satochip.io)做第二数据点 |
| 非接读卡器 | PC/SC,支持 ISO14443(如 ACR1252U);接触式做 GP 装载(如 ACR39) | 常规渠道 |
| 手机 | iPhone(XS 或更新)、支持 NFC 的 Android 各 1 | 现有即可 |

**下单前务必确认(避坑)**:卡必须是**未熔断/GP Card Manager 可选中**,且卖家提供 **GP 传输密钥/默认测试密钥**(否则装不进 applet)。默认 GP 测试密钥常见为 `404142434445464748494A4B4C4D4E4F`。**别买 AliExpress/eBay 上不说明密钥状态的"神秘"JCOP 卡。**

## B. 量产询价(向 SE 芯片代理 + 深圳金融 IC 卡厂各发一份)

### B1. 芯片(向 NXP 代理 / 授权分销)

- **型号**:NXP JCOP4 系列,**J3R200 / P71 家族**(JavaCard 3.0.5 Classic,GlobalPlatform 2.3,EAL6+,双界面 contactless)。若有更大 flash 版本(如 200K)一并报。
- **要问**:
  1. 各量级档位单价:**1k / 5k / 10k / 50k / 100k / 500k**;
  2. 供货周期、MOQ、是否需签署任何协议;
  3. 是否原生支持:`ALG_HMAC_SHA_256/512`、`ALG_EC_SVDP_DH_PLAIN(_XY)`、`TYPE_EC_FP_PRIVATE_TRANSIENT_DESELECT`、任意域参数 EC(secp256k1);
  4. (记录用,预期"否")是否原生 `ALG_EDDSA`(ed25519)。

### B2. 成品卡(向深圳金融 IC 卡产线 / 卡厂)

- **规格**:双界面(接触 + ISO14443 非接)JavaCard 卡,内置上面的 NXP JCOP4 P71 芯片;标准卡尺寸(CR80);可数码印卡(每场活动限定卡面 + 编号);防拆封装。
- **服务范围要问清**:
  1. 是否由卡厂**采购芯片 + 封装成卡**,还是我方供芯片;
  2. 是否提供 **applet 装载(GP LOAD/INSTALL 我方 CAP)** 与 **个人化(perso)**:密钥注入、写序列号、印卡面;
  3. **perso 安全能力**:能否支持"我方 HSM 生成密钥 → 密文注入包 → 卡厂只装密文、不见明文"的流程;密钥仪式/分权规程;
  4. 产线是否有**金融 IC 卡 / POS 安全生产资质**;
  5. **小批量数码印卡**能力(每场活动 500–2000 张限定卡面)与其加价;
  6. 各量级档位**成品单价**(含芯片 + 卡 + perso + 印刷):**500 / 1k / 5k / 10k / 50k / 100k**;
  7. 打样周期 + 量产周期 + MOQ。
